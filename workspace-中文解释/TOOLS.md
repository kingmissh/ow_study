# TOOLS.md - 本地笔记

技能定义工具如何工作。这个文件是关于你的具体情况——你的设置特有的东西。

## 这里放什么

诸如此类的东西：

- 摄像头的名称和位置
- SSH 主机和别名
- TTS 首选语音
- 扬声器/房间名称
- 设备昵称
- 任何特定于环境的东西

## 示例

```markdown
### 摄像头

- living-room → 主区域，180° 广角
- front-door → 入口，运动触发

### SSH

- home-server → 192.168.1.100，用户：admin

### TTS

- 首选语音："Nova"（温暖，略偏英式）
- 默认扬声器：厨房 HomePod
```

## 为什么分开？

技能是共享的。你的设置是你的。保持它们分开意味着你可以更新技能而不丢失你的笔记，分享技能而不泄露你的基础设施。
